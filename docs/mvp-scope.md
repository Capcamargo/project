# MVP scope

## Что входит в MVP
- форма подбора подарка;
- ввод повода, бюджета и интересов получателя;
- генерация или показ персонализированной подборки;
- карточки подарков с кратким описанием;
- переход к покупке или сохранению выбранного варианта.

## Что пока не входит в MVP
- корпоративный кабинет;
- собственная платежная система;
- сложная логика доставки внутри платформы;
- подписка и премиум-функции;
- личный кабинет с историей заказов;
- напоминания о датах и праздниках;
- широкая партнерская сеть с локальными магазинами.

## Ключевые функции
1. Быстрый подбор подарка по нескольким параметрам.
2. Персонализированные рекомендации.
3. Понятные карточки вариантов.
4. Возможность быстро перейти к дальнейшему действию.

## User flow
1. Landing page
2. CTA «Подобрать подарок»
3. Форма подбора
4. Экран результатов
5. Переход к покупке / сохранению shortlist

## Критерий успешности MVP
Пользователь должен пройти путь от входа на страницу до получения подборки без лишних шагов и понять, какой вариант подарка выбрать дальше.
