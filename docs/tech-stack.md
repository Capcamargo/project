# Технический стек GiftMatch

## Frontend / mobile
### TypeScript
TypeScript помогает удерживать структуру проекта в порядке и снижает число ошибок по мере роста MVP.

### React Native
React Native подходит для быстрой сборки мобильного интерфейса без раздельной разработки под каждую платформу.

### Expo
Expo упрощает старт проекта, ускоряет разработку и хорошо подходит для первых итераций MVP.

## Backend / data
### Supabase
Supabase закрывает базовые задачи backend-слоя: база данных, авторизация и работа с данными через API.

## Почему этот выбор подходит GiftMatch
GiftMatch — это не heavy backend-продукт на старте, а сервис с фокусом на одном главном пользовательском сценарии. Поэтому стек должен помогать быстро доводить идею до работающего прототипа, а не перегружать проект инфраструктурой.
