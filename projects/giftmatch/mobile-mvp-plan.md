# Mobile MVP plan

## Цель
Собрать первую мобильную версию GiftMatch вокруг одного главного сценария: пользователь указывает повод, бюджет и интересы получателя, а затем получает подборку подарков.

## Базовые экраны
1. Экран приветствия.
2. Экран формы подбора.
3. Экран результатов.
4. Экран профиля пользователя.

## Первый релиз
В первом релизе достаточно реализовать форму подбора и экран результатов. Остальные части можно добавлять следующими итерациями.
