# GiftMatch MVP

Папка с веб-версией сервиса персонализированного подбора подарков.

## Основная версия сайта
- `app.html` — главная пользовательская версия сайта;
- `pricing.html` — страница тарифного шага;
- `exam.css` — текущие стили интерфейса;
- `exam.js` — логика сценария и интерактивности.

## Дополнительные версии
- `exam.html` — альтернативная сборка на той же логике;
- `exam-success.html` — альтернативная страница тарифного шага;
- `index.html` — ранняя demo-версия GiftMatch;
- `style.css` — стили ранней demo-версии;
- `script.js` — логика ранней demo-версии;
- `success.html` — ранний экран тарифного шага.

## Проектные документы
- `open-local.md` — как открыть проект локально;
- `demo-scenario.md` — порядок показа основного сценария;
- `presentation-outline.md` — основа для короткой презентации;
- `mobile-mvp-plan.md` — план мобильной версии;
- `user-flow.md` — пользовательский сценарий;
- `screens.md` — экранная структура;
- `components-map.md` — карта компонентов;
- `release-plan.md` — этапы развития;
- `supabase/` — SQL-схема и связанные материалы по данным.

## Что уже умеет сайт
1. Показывать ценность сервиса на первом экране.
2. Создавать пользовательский профиль.
3. Принимать параметры подбора подарка.
4. Строить первую подборку с пояснениями.
5. Сохранять подборки в истории пользователя.
6. Ограничивать число сохранений на бесплатном плане.
7. Показывать следующий тарифный шаг отдельной страницей.

## Что открывать
Для локального просмотра и дальнейшей доработки лучше использовать `app.html`.
