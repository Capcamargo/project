# GiftMatch MVP

Папка с веб-версией MVP сервиса персонализированного подбора подарков.

## Главная версия для показа
- `exam.html` — основная качественная экзаменационная версия сайта;
- `exam.css` — стили для экзаменационной версии;
- `exam.js` — логика сценария для экзаменационной версии;
- `exam-success.html` — безопасный экран демонстрации оплаты.

## Дополнительные файлы
- `index.html` — ранняя demo-версия GiftMatch;
- `style.css` — стили ранней demo-версии;
- `script.js` — логика ранней demo-версии;
- `success.html` — ранний экран демонстрации оплаты;
- `open-local.md` — как открыть проект локально;
- `demo-scenario.md` — порядок показа на защите;
- `presentation-outline.md` — основа для короткой презентации;
- `mobile-mvp-plan.md` — план мобильной версии;
- `user-flow.md` — пользовательский сценарий;
- `screens.md` — экранная структура;
- `components-map.md` — карта компонентов;
- `release-plan.md` — этапы развития;
- `supabase/` — SQL-схема и связанные материалы по данным.

## Что показывает экзаменационная версия
1. Первый экран с ценностью сервиса.
2. Создание demo-профиля.
3. Заполнение формы подбора.
4. Получение объяснимых рекомендаций.
5. Сохранение подборок.
6. Ограничение Free-сценария.
7. Отдельный демонстрационный экран оплаты без автоматического открытия premium.

## Рекомендуемый способ открытия
Для защиты и показа лучше использовать именно `exam.html`.

## Главная цель этой версии
Показать на защите один убедительный end-to-end сценарий: от входа пользователя в продукт до получения, объяснения и сохранения результата с демонстрацией paywall.
