# GiftMatch MVP

Папка проекта для дальнейшей реализации MVP сервиса персонализированного подбора подарков.

## Что здесь будет
- клиентская часть MVP;
- конфигурация проекта;
- подключение Supabase;
- дальнейшие экраны и логика подбора.

## Текущая структура
- `supabase/` — SQL-схема и материалы по базе данных.
