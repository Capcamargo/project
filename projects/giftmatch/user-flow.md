# User flow

## Основной сценарий
1. Пользователь открывает GiftMatch.
2. Видит понятное ценностное предложение.
3. Нажимает кнопку начала подбора.
4. Заполняет короткую форму.
5. Получает персонализированную подборку.
6. Выбирает вариант для просмотра или покупки.

## Ключевой критерий
Пользователь должен получить первый полезный результат без лишних шагов и перегруженного интерфейса.
