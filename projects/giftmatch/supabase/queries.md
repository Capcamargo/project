# Supabase queries plan

## Основные операции

### 1. Создание профиля
Создается после регистрации пользователя и используется для хранения базовых данных аккаунта.

### 2. Сохранение gift request
После отправки формы подбора создается запись с поводом, бюджетом и интересами получателя.

### 3. Сохранение recommendations
После формирования результатов рекомендации могут сохраняться в отдельной таблице и привязываться к конкретному запросу пользователя.

### 4. Чтение своих данных
Пользователь видит только собственные запросы и собственные рекомендации.

## Что проверить при подключении
- корректное создание профиля;
- сохранение запроса;
- получение результатов;
- отсутствие доступа к чужим данным.
