# Supabase

Материалы для локальной и облачной базы данных GiftMatch.

## Структура
- `config.toml` — локальная конфигурация Supabase CLI;
- `migrations/202605150001_init_giftmatch.sql` — основная миграция со схемой проекта;
- `seed.sql` — стартовые данные для пресетов GiftMatch;
- `schema.sql` — плоская версия схемы для ручного запуска в SQL Editor;
- `queries.md` — список ключевых запросов и сценариев;
- `rls-notes.md` — заметки по ограничениям доступа;
- `sample-data.sql` — пример тестовых записей.

## Как использовать локально
1. Открыть папку `projects/giftmatch/supabase/`.
2. Запустить Supabase CLI в режиме local development.
3. Применить миграции из `migrations/`.
4. Загрузить стартовые данные из `seed.sql`.
5. После этого подключать фронтенд к локальной или облачной базе.

## Как использовать без локального Supabase
1. Открыть Supabase Studio или SQL Editor в облачном проекте.
2. Выполнить SQL из `schema.sql`.
3. При необходимости отдельно выполнить вставки из `seed.sql`.

## Что уже покрывает схема
- профили пользователей;
- запросы на подбор подарка;
- результаты рекомендаций;
- публичные пресеты для быстрого старта;
- RLS для пользовательских данных.
