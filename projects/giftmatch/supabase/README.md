# Supabase

Материалы для подключения базы данных к GiftMatch.

## Содержимое
- `schema.sql` — базовая SQL-схема для таблиц профилей пользователей, запросов на подбор и результатов рекомендаций.

## Как использовать
1. Открыть Supabase.
2. Перейти в SQL Editor.
3. Выполнить SQL из `schema.sql`.
4. После этого подключать проект к Supabase из MVP.
