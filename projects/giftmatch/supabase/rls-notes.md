# RLS notes

## Зачем нужны ограничения
GiftMatch работает с пользовательскими запросами и персональными рекомендациями. Даже для MVP важно, чтобы пользователь видел только свои данные.

## Что уже заложено в схеме
- профиль доступен только владельцу;
- запросы на подбор доступны только владельцу;
- рекомендации доступны только владельцу.

## Что это дает
RLS помогает не полагаться только на клиентскую логику и снижает риск случайного или неправильного доступа к чужим данным.
