# GiftMatch

GiftMatch — учебный MVP сервиса персонализированного подбора подарков по интересам, бюджету и поводу.

## О проекте
Проект решает простую и понятную проблему: людям сложно быстро выбрать уместный подарок без долгого поиска по разным магазинам и маркетплейсам. GiftMatch помогает сократить время выбора и сделать рекомендации более персонализированными.

## Проблема
- сложно придумать оригинальный и подходящий подарок;
- поиск по разным площадкам занимает слишком много времени;
- есть риск выбрать неудачный подарок;
- пользователю нужен более быстрый и понятный путь к решению.

## Целевая аудитория MVP
Первый MVP ориентирован на занятых пользователей 18–35 лет, которые выбирают подарок для друга, партнера, родственника или коллеги и хотят сделать это быстро и без стресса.

## Ценность продукта
Пользователь отвечает на несколько простых вопросов и получает персонализированную подборку подарков. Это снижает стресс, экономит время и делает выбор более осознанным.

## Основной JTBD
Когда мне нужно выбрать подарок близкому человеку и я не уверен, что ему подойдет, я хочу за несколько минут получить уместные идеи с учетом интересов, повода и бюджета, чтобы быстро выбрать хороший подарок без стресса и долгого поиска.

## MVP scope
В первую версию входят:
- форма подбора подарка;
- ввод повода, бюджета и интересов получателя;
- экран результатов;
- карточки подарков с кратким описанием;
- переход к дальнейшему действию.

## Выбранный стек
### Frontend / mobile
TypeScript + React Native + Expo

### Backend / data
Supabase

## Почему именно этот стек
- один стек и одна кодовая база для дальнейшего развития под iOS и Android;
- быстрый старт для MVP;
- удобно использовать вместе с AI-инструментами;
- Supabase позволяет быстро получить базу данных, API и базовую backend-инфраструктуру без собственного сервера с нуля.

## Базовый user flow
1. Пользователь открывает лендинг.
2. Нажимает кнопку начала подбора.
3. Заполняет форму.
4. Получает подборку подарков.
5. Переходит к просмотру или покупке подходящего варианта.
