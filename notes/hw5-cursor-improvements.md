# HW5 — Улучшения MVP через Cursor

## Цель
Аккуратно доработать проект GiftMatch с помощью Cursor и AI-подсказок без потери основной логики MVP.

## Что важно улучшить
- сделать структуру проекта понятнее;
- сократить лишние элементы интерфейса;
- улучшить читаемость текстов на экранах;
- не усложнять сценарий подбора подарка;
- сохранять фокус на одном основном пользовательском действии.

## Практический результат
После доработок MVP должен оставаться простым, понятным и удобным для дальнейшего развития.
