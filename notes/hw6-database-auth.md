# HW6 — Подключение базы данных и авторизация

## Что сделано для GiftMatch
Для проекта подготовлена Supabase-схема с таблицами:
- `profiles` — профили пользователей;
- `gift_requests` — анкеты подбора подарков;
- `gift_recommendations` — результаты рекомендаций.

## Что дает Supabase
- база данных PostgreSQL;
- Auth для регистрации и входа;
- API без собственного backend с нуля.

## Что нужно проверить при реализации
1. Пользователь может зарегистрироваться и войти.
2. Пользователь может сохранить запрос на подбор подарка.
3. Пользователь видит только свои запросы и свои рекомендации.
4. Доступ к данным ограничен через RLS.
