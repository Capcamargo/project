# HW3 — Лендинг и интерактивный сценарий

## Цель
Собрать первый экран GiftMatch и простой пользовательский сценарий подбора подарка.

## Что должно быть на лендинге
- понятный заголовок;
- краткое описание ценности сервиса;
- CTA-кнопка начала подбора;
- блок «Как это работает»;
- преимущества сервиса;
- форма начала подбора подарка.

## Основной интерактивный сценарий
1. Пользователь открывает лендинг.
2. Нажимает кнопку начала подбора.
3. Заполняет повод, бюджет и интересы получателя.
4. Получает экран с персонализированными вариантами подарков.

## Что важно для MVP
Лендинг должен не просто рассказывать о проекте, а доводить пользователя до первого полезного действия — начала подбора подарка.
