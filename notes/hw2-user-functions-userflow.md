# HW2 — Пользователь, функции и user flow

## Основной пользователь
Основной пользователь MVP — человек, которому нужно быстро выбрать подарок близкому человеку, но он не уверен, что именно подойдет.

## Основной JTBD
Когда мне нужно выбрать подарок близкому человеку и я не уверен, что ему подойдет, я хочу за несколько минут получить уместные идеи с учетом интересов, повода и бюджета, чтобы быстро выбрать хороший подарок без стресса и долгого поиска.

## Ключевые функции MVP
1. Форма подбора подарка.
2. Ввод повода, бюджета и интересов получателя.
3. Персонализированная подборка рекомендаций.
4. Карточки подарков с кратким описанием.
5. Переход к дальнейшему действию.

## User flow
1. Пользователь заходит на лендинг.
2. Нажимает CTA-кнопку начала подбора.
3. Заполняет короткую форму.
4. Получает персонализированную подборку подарков.
5. Выбирает подходящий вариант.
6. Переходит к просмотру или покупке.
