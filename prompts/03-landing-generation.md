# Prompt 03 — Landing generation

Ты UX-копирайтер и продуктовый дизайнер.
Сгенерируй структуру одностраничного лендинга для сервиса персонализированного подбора подарков.

Нужны блоки:
- заголовок;
- подзаголовок;
- CTA-кнопка;
- как это работает;
- преимущества;
- форма начала подбора.

Контекст:
Сервис помогает быстро подобрать подарок по интересам, поводу и бюджету.

Формат ответа:
Понятная структура лендинга и короткие тексты для первого экрана.
