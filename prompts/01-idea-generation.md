# Prompt 01 — Idea generation

Ты продуктовый аналитик.
Помоги сформулировать идею MVP сервиса персонализированного подбора подарков.

Нужно выдать:
1. проблему;
2. целевую аудиторию;
3. ценность продукта;
4. краткое описание идеи.

Контекст:
Сервис помогает подобрать подарок по интересам, бюджету и поводу.
Цель — сделать первый понятный MVP без лишней сложности.

Формат ответа:
Коротко, структурно, без бизнес-плана и без лишней теории.
