# AI prompts

## Промпт для описания MVP
Ты продуктовый аналитик. Помоги сформулировать MVP сервиса персонализированного подбора подарков.
Нужно выдать:
1. проблему;
2. целевую аудиторию;
3. ценность продукта;
4. 3–5 ключевых функций;
5. краткий пользовательский путь.
Формат ответа: коротко, структурно, без лишней теории.

## Промпт для генерации лендинга
Ты UX-копирайтер и продуктовый дизайнер.
Сгенерируй структуру одностраничного лендинга для сервиса персонализированного подбора подарков.
Сервис помогает подобрать подарок по интересам, бюджету и поводу.
Нужны блоки:
- заголовок;
- подзаголовок;
- CTA-кнопка;
- как это работает;
- преимущества;
- форма начала подбора.
Сделай текст понятным, коротким и ориентированным на первое касание пользователя.

## Промпт для генерации first MVP screen
Ты продуктовый дизайнер.
Опиши экран формы подбора подарка.
На экране должны быть:
- повод;
- бюджет;
- интересы получателя;
- степень близости;
- кнопка получения рекомендаций.
Формат ответа: структура экрана, подписи элементов, логика взаимодействия.
