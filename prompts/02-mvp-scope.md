# Prompt 02 — MVP scope

Ты продуктовый менеджер.
Помоги описать MVP сервиса персонализированного подбора подарков.

Нужно выдать:
1. основного пользователя;
2. основной JTBD;
3. 3–5 ключевых функций MVP;
4. user flow;
5. что входит в MVP, а что пока не входит.

Контекст:
Сервис подбирает подарки по поводу, интересам и бюджету.
Нужно сфокусироваться на первом рабочем сценарии, а не на полном бизнесе.

Формат ответа:
Коротко, структурно, понятно для документации проекта.
